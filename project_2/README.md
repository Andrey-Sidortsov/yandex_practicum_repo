# Нейросеть для автодополнения текстов

## Описание проекта
Сравнение архитектур LSTM, GRU и предобученного трансформера distilgpt2 для задачи предсказания следующего слова.  
Датасет: WikiText-2.

## Цель
Определить оптимальную модель для использования в мобильном мессенджере с учётом ограничений по размеру и скорости.

## Архитектуры
| Модель | Параметров | Время на эпоху (GPU) |
|--------|------------|---------------------|
| LSTM   | ~3.5M      | ~11 мин             |
| GRU    | ~3.4M      | ~10 мин             |
| distilgpt2 | ~82M   | - (предобучена)     |

## Результаты
| Модель | Test Perplexity | Размер |
|--------|-----------------|--------|
| LSTM   | 370.53          | ~3.5M  |
| GRU    | 265.31          | ~3.4M  |
| distilgpt2 | 94.20     | ~82M   |

## Выводы
**GRU** — оптимальный выбор для мобильного устройства:
- Минимальный размер модели
- Достаточное качество для подсказок
- Быстрая работа на устройстве
